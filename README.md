SOR files (Standard OTDR Record) are a specialized file format used in telecommunications. If you encounter one, it's likely related to fiber optic networks.
My program is based on the pyotdr project - https://github.com/sid5432/pyOTDR
Main functions: opening multiple files, displaying a graph, working with a graph, and displaying event data in a table.
Technologies:
Version Python - 3.13
Version Qt - 6
